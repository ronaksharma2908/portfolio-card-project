# 💼 Portfolio Card Project

A simple and elegant **React portfolio card** component built with **Create React App**.  
This project demonstrates reusable components, modular CSS, and clean UI design for showcasing personal information in a card format.

---

## ✨ Features
- **Reusable Components**: Avatar, Bio, Header, and Personal Info.  
- **Responsive Design**: Works across desktop and mobile.  
- **Custom Styling**: Modular CSS (`portfoliocard.css`) for easy customization.  
- **React Best Practices**: Component‑based architecture with clear separation of concerns.  

---

## 🛠️ Tech Stack
- **React** (via Create React App)  
- **JavaScript (ES6+)**  
- **CSS Modules**  
- **Node.js** (for development environment)
